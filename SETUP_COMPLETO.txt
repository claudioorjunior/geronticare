## 🚀 Próximos Passos

### 1. Configurar banco de dados

Escolha uma opção:

**A) Neon (recomendado - grátis):**
1. Acesse https://neon.tech
2. Crie projeto 'geronticare'
3. Copie connection string
4. Cole no .env.local:
   DATABASE_URL="postgresql://user:pass@host.neon.tech/geronticare"

**B) Supabase:**
1. Acesse https://supabase.com
2. Crie projeto 'geronticare'
3. Database → Connection string (Transaction mode)
4. Cole no .env.local

**C) PostgreSQL Local:**
brew install postgresql@16
brew services start postgresql@16
createdb geronticare
DATABASE_URL="postgresql://seu-user@localhost:5432/geronticare"

### 2. Aplicar migrations
cd ~/Documents/geronticare
npm run db:push

### 3. Iniciar desenvolvimento
npm run dev
# Acesse http://localhost:3000

### 4. Publicar no GitHub
git init
git add .
git commit -m "feat: initial commit - GerontiCare MVP"
gh repo create geronticare --public --source=. --push
